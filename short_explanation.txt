A gamified self-improvement web application where users embark on a spiritual journey, guided by a mentor spirit to combat negative influences through daily activities and decision-making.

Core Features:
- User authentication system for tracking individual progress and journey
- "Face-off" system where users log their daily decisions and challenges through guided logic flowcharts
- Progress tracking for essential daily activities (like walking/running one mile)
- Spirit guide interaction system with advice and decision-making flow charts for different life scenarios (relationships, health, habits)

Visual References:
Inspired by RPG games like Stardeer Valley and wellness apps like Headspace, combining gamified elements with mindful, spiritual aesthetics.

Style Guide:
- Colors: Primary #7B61FF (spiritual purple), Secondary #34C759 (positive green), Warning #FF6B6B (demon red), Background #F5F7FA (serene white), Accent #FFD93D (spirit gold), Text #2C3E50 (deep wisdom)
- Design: Poppins/Lato fonts, mystical yet modern interface with card-based layouts, soft shadows, nature-inspired elements, progress indicators, and achievement badges, 16px spacing, responsive design with smooth transitions
